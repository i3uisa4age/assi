TATHA
เว็บไซต์สำหรับซื้อขาย อุปกรณืบาสเกตบอลที่มีคติคือ "ของไทยเพื่อคนไทย" ให้บริการด้านการบริการมีทั้งออนไลน์และหน้าร้าน 

ประกอบด้วย:

วิสัยทัศน์: เพื่อให้บริการอุปกรณ์กีฬาและชุดกีฬาคุณภาพสูงสำหรับนักกีฬาทุกระดับ

บริการ: ขายอุปกรณ์ทุกชนิดที่เกี่ยวข้องกับบาสเกตบอล

โครงสร้างราคา: ตารางเปรียบเทียบราคาสินค้า

ทีมผู้บริหาร: ข้อมูลบุคลากรขององกรณ์

การติดต่อ: แบบฟอร์มติดต่อภารกิจและแผนที่ที่ตั้งบริษัท
